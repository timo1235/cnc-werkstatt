Table Router Shield - Firmware Installer

Flashe deinen ESP32 direkt im Browser. Keine Software-Installation noetig.

Table Router Shield v2.0

Verbinde deinen ESP32 per USB und klicke auf den Button.

Dein Browser unterstuetzt kein Web Serial. Bitte verwende Google Chrome oder Microsoft Edge. Diese Seite muss ueber HTTPS bereitgestellt werden.
Voraussetzungen:
  • Google Chrome oder Microsoft Edge (Desktop)
  • ESP32 per USB-Kabel verbunden
  • USB-Treiber installiert (CP2102 oder CH340)
Fehlerbehebung:
  • Falls kein Port angezeigt wird, installiere den passenden USB-Treiber (siehe oben).
  • Halte den BOOT-Button am ESP32 gedrueckt, waehrend du das USB-Kabel ansteckst, falls das Flashen fehlschlaegt.
